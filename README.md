# Experiential Framework for databinding using ES6 Pro

## Requirements

- [Node and npm](http://nodejs.org)

## Installation

1. `git clone git@github.com:oriweingart/ProxyComponent`
2. `cd ProxyComponent`
3. `npm install`
3. `npm start`
4. Go to `http://localhost:3000`
5. Play with app.clinet.js
